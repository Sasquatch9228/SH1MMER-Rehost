Instructions

See the directions at home on how to find a shim for your device. If there isn't a shim for your device, then good luck finding one.. somehow.

Once you've found your shim, make sure to extract it if it's in a .zip file. If it has any other kind of compression (.gz, .bz2, .xz, etc) make sure you decompress the file first.

Before pressing the start button, you may select or deselect any of the options below.

Next, hit the start button and select your .bin file. The VM will start up. This will use a considerable amount of memory on your system and take a while. If it freezes, that's normal. Once it finishes, it will automatically download the patched version.
